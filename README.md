# eBook Evolution - CIMA Embedded App (Vanilla JS + Netlify Functions)

A vanilla JS single page app you can embed in CIMA.

## Quick Start
1. Install Netlify CLI (`npm install -g netlify-cli`)
2. Run locally: `netlify dev`
3. Deploy to Netlify and set environment variables.

## Features
- HTML chapter rendering
- Search, highlighting, notes
- Accessibility tools (speech, contrast, font resize)
- Responsive layout
- Progress and completion tracking
- Permissions and DRM ready (future)
